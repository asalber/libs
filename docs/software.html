Software para el análisis de datos espectroscópicos

Paquetes de R

Paquete hyperSpec

El paquete hyperSpec es un paquete de R que proporciona herramientas para la manipulación y análisis de datos espectroscópicos. Este paquete permite trabajar con datos espectroscópicos de forma sencilla y eficiente, y proporciona herramientas para visualizar y analizar espectros ópticos.

  • Introducción a hyperSpec

Paquete resemble

El paquete resemble proporciona herramientas para extraer de manera eficiente y precisa información cuantitativa significativa de bases de datos espectrales grandes y complejas. Las funcionalidades principales del paquete incluyen: reducción de dimensionalidad, cálculo de medidas de disimilitud, evaluación de matrices de disimilitud, búsqueda de espectros similares, ajuste y predicción de modelos espectroscópicos locales.

  • Introducción a resemble

Paquete prospectr

El paquete prospectr es un paquete de R que proporciona herramientas para el análisis de datos espectroscópicos. Este paquete está especializado en el preprocesamiento de datos espectroscópicos y proporciona herramientas para la corrección de ruido, la normalización de datos y el calibrado de la muestra.

  • Introducción a prospectr

Paquete Spectral Analysis

El paquete Spectral Analysis es otro paquete de R que proporciona herramientas para el análisis de datos espectroscópicos. Permite realizar las tareas más comunes en el análisis de espectros, como la corrección de ruido, la normalización de datos y la identificación de picos.

  • Introducción a Spectral Analysis

Paquetes de Python

Paquete SimulatedLIBS

El paquete SimulatedLIBS es un paquete de Python que permite simular espectros de emisión de un plasma inducido por láser (LIBS) a partir de la composición química de una muestra.

  • Documentación

Paquetes de Julia

Paquete Spectra

El paquete Spectra es un paquete de Julia para el análisis de datos espectroscópicos que permite realizar las tareas más comunes en el análisis de espectros de absorción y emisión, como ajuste de la línea base, ajuste de picos, integración de picos, cálculo de áreas, etc.
